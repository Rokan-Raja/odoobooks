Odoo 10 Best Practices
======================
This book is dedicated for entrepreneur who wants to implement all-in-one
business application out-of-the box for their business.

**Odoo 10 Best Practices Book** is an online book written based on Community and Enterprise
Edition. This book covers best implementation practice for most used
applications such as Sales, Purchase, Inventory, Financial Accounting and
Manufacturing.

.. warning:: Odoo 10.0 is unsupported with the release of Odoo 13.0, Odoo supports
  last 3 LTS version only. You should migrate to upper version with
  `Odoo Upgrade Service <https://upgrade.odoo.com/database/upload>`_

.. note:: You can subscribe to `Youtube <https://www.youtube.com/c/MantavyaGajjar>`_
  channel to receive an instance update about the new videos and pages on this book.
